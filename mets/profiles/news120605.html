METS News and Announcements

New Version 1.5 of METS Schema Released (December 6, 2005)

A revised version (Version 1.5) of the METS XML schema is now available at the official METS web site at http://www.loc.gov/standards/mets/mets-schemadocs.html. Please note that this revision requires the use of an additional XML schema document defining XLink attributes used in the METS schema. The XLink schema is available at http://www.loc.gov/standards/mets/xlink.xsd

This revision makes a number of changes and enhancements to the METS schema, including:

  • Made file element recursive to deal with PREMIS Onion Layer model and support XFDU-ish unpacking specification.
  • Add element beneath to allow linking of metadata to subfile structures.
  • Modify structLink TO and FROM attributes to put them in XLink namespace.
  • Make processContents "lax" for all xsd:any elements.

Input from the library community regarding the design of the METS schema is encouraged. Comments regarding METS' design should be submitted to the METS listserv. Details about subscribing to the listserv are available at the METS web site).
